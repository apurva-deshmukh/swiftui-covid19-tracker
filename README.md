# swiftui-covid19-tracker
SwiftUI COVID-19 tracker with detailed country statistics and a map that shows confirmed cases and deaths for each country

## APIs Used
  * covid-193.p.rapidapi.com
  * covid-19-data.p.rapidapi.com

## Statistics
  * Total
    * Confirmed Cases
    * Critical Cases
    * Deaths
    * Recovered
    * Fatality Rate
    * Recovery Rate
  * Country
    * Confirmed Cases
    * New Cases
    * Recovered
    * Critical Cases
    * Active Cases
    * Deaths
    * New Deaths
    * Tests Done
    
 ## Swift Libraries
  * Alamofire
  * SwiftyJSON
